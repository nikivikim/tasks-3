# Медиатор


Приложение с реализацией паттерна Медиатор.

**Описание**
---
Это программный модуль авторизации в системе тестирования с тремя уровнями доступа:
- Студент. Отвечает на вопрсы теста;
- Преподаватель. Создает, редактирует и размещает тестовые задания:
- Посетитель. Может только посмотреть тестовые задания.

**Функционал приложения:**

- Возможность авторизации;
- Решение теста:
- Редактирование теста;
- Просмотр содержимого;
- Сохранение изменений.


**Технология**
---
Язык программирования Java.
